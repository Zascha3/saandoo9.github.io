Football performance through country indicators

This page is designed as a connected visual story. Each section combines a short written interpretation with a plot placed next to the text, so the reader can move from explanation to evidence without leaving the page.

Page structure

  • Section 1 introduces the interactive correlation plot.
  • Sections 2, 3 and 4 are ready for three additional visualizations.
  • All plots are loaded from the visualizations/ folder.
Section 1

Do country indicators explain football outcomes?

The first visualization explores whether socioeconomic indicators are associated with football performance. The user can switch between football metrics and country-level factors to compare the strength and direction of each relationship.

This plot should not be read as causal evidence. Instead, it works as an exploratory tool: it helps identify which variables show stronger patterns, which countries behave as outliers, and where the relationship appears weak or noisy.
